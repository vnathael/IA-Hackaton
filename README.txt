# Projet Hackathon - Le Plastique dans l'Océan

## Description

Ce projet fait partie d'un hackathon ayant pour thème "Le plastique dans l'océan". Le jeu se déroule dans un environnement 2D où le joueur incarne un animal marin (tortue, calmar ou poisson) qui doit éviter des déchets plastiques. Le but est de survivre le plus longtemps possible tout en apprenant des faits sur la pollution plastique et ses impacts sur la faune et l'environnement.

## Lancer le projet

1. Ouvrir le projet dans **Visual Studio Code**.
2. Localiser et faire un clic droit sur le fichier `index.html`.
3. Sélectionner **"Open with Live Server"** pour lancer le jeu dans votre navigateur.

## Règles du jeu

### Objectif

Il y a trois jeux disponibles avec différents animaux marins, chacun ayant des problèmes avec le plastique dans l’océan. Le but du jeu est d'éviter les déchets plastiques aussi longtemps que possible.

### Commandes

- Utilise les **flèches du clavier** pour déplacer ton animal à travers l'écran.
- Il y a **trois animaux** :
  - **Tortue**
  - **Calmar**
  - **Poisson**
  
  Chaque animal a des caractéristiques et des vitesses différentes.

### Mécanisme du jeu

- À chaque **3 collisions** avec des déchets plastiques, un **popup** s'affiche avec :
  - Une **cause de la pollution plastique**,
  - Un **effet sur la faune ou l'environnement**,
  - Ou une **solution possible** pour contrer cette pollution.

### Difficulté

- La difficulté augmente au fur et à mesure du jeu. Les déchets deviennent plus rapides et apparaissent plus fréquemment.
- Le jeu continue jusqu'à ce que l'animal meure en touchant un déchet plastique.

## Packages et Dépendances

Ce projet utilise les technologies suivantes :

- **HTML** : Structure de la page et du jeu.
- **CSS** : Pour le style de l'interface et des éléments du jeu.
- **JavaScript** : Logique du jeu, gestion des collisions et de l'augmentation de la difficulté.

### Aucun package externe n'est nécessaire pour ce projet. Tout est inclus dans les fichiers HTML, CSS et JavaScript.
